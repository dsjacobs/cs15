


ADD
- by index (insertAt)

- in alpha (ascii) order (insertInOrder)

- by index (removeAt)
- from front or back (popFromFront and popFromBack)

CHECK
- for empty (isEmpty)
- size (size)

RETURN
- first (first)
- last (last)
- by index (elementAt)

- update element at index (replaceAt)
